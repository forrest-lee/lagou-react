| 路由                       | 含义                  |
| ------------------------- |:---------------------:|
| /api/v1/position/all      | 返回所有职位的摘要要信息  |
| /api/v1/position/:id      | 返回某个职位的详细信息    |
| /api/v1/keywords          | 返回关键词列表           |
| /api/v1/user              | 返回用户信息             |
| /api/v1/recommend         | 返回与用户偏好匹配的职位   |
